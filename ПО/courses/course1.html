1. Кіріспе

MySQL — ашық кодты реляциялық мәліметтер қоры жүйесі (RDBMS). Ол деректерді сақтау және басқару үшін қолданылады.

Орнату қадамдары

sudo apt update
sudo apt upgrade
sudo apt install mysql-server

Негізгі командалар

  • SHOW DATABASES; — Мәліметтер қорын көру
  • CREATE DATABASE name; — Жаңа қор құру
