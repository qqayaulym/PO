2. Орнату

MySQL-ді орнату үшін төмендегі қадамдарды орындаңыз:

  1. MySQL-ді жүктеңіз
  2. Орнату процесін аяқтаңыз
  3. Бастапқы конфигурацияны орнатыңыз

Егер сіз Ubuntu қолдансаңыз, терминалда келесі командаларды орындаңыз:

sudo apt update
sudo apt upgrade
sudo apt install mysql-server

Орнату аяқталғаннан кейін, MySQL қызметін іске қосыңыз:

sudo systemctl start mysql

MySQL-дің дұрыс орнатылғанын тексеру үшін келесі команданы орындаңыз:

sudo systemctl status mysql
