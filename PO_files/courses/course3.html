3. SELECT негіздері

SELECT операторы MySQL-да мәліметтерді сұрау үшін қолданылады. Бұл оператор кестеден деректерді тауып, оларды көрсету үшін пайдаланылады.

SELECT операторын қолдану үшін келесі синтаксис қолданылады:

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Мысалы, "students" кестесінен барлық студенттердің атын және жасын алу үшін келесі сұранысты қолдануға болады:

SELECT name, age
FROM students;

WHERE шартын қолдану арқылы белгілі бір критерийлерге сәйкес жазбаларды сұрауға болады. Мысалы, жасы 20-дан үлкен студенттердің атын алу үшін:

SELECT name
FROM students
WHERE age > 20;
