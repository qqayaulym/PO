- `/cart`, `/course/:id`, `/admin` беттеріне рұқсатсыз кіру мүмкін болған

Түзету:

- [frontend/src/App.jsx](/C:/Users/Газиза/Desktop/PO/frontend/src/App.jsx) ішінде `RequireAuth` логикасы жаңартылды
- Логинсіз user `/signin` бетіне жіберіледі
- `adminOnly` беттері role бойынша қорғалады

Код үзіндісі:

```jsx
function RequireAuth({ currentUser, adminOnly = false, children }) {
  if (!currentUser) {
    return <Navigate to="/signin" replace />;
  }

  if (adminOnly && currentUser.role !== "admin") {
    return <Navigate to="/" replace />;
  }

  return children;
}
```

Қазіргі нәтиже:

- Дұрыс логин/пароль жұмыс істейді
- Қате пароль кезінде қате хабарлама шығады
- Reload-тан кейін сессия сақталады
- Логинсіз қолданушы қорғалған беттерге кіре алмайды
- Admin емес қолданушы `/admin` бетіне кіре алмайды
